# Daily Task Journal

A simple and intuitive task management web application built with **HTML, CSS and JavaScript**.
The project focuses on practicing **DOM manipulation, state management and browser storage** while delivering a clean and responsive user experience.

## Features

* Add, edit and delete tasks
* Mark tasks as completed
* Task filtering (All / Active / Completed)
* Real-time task search
* Dark and light theme support
* Task creation date
* Progress tracking with a dynamic progress bar
* Task counter
* Clear all tasks or only completed tasks
* Smooth UI feedback animations
* Persistent data using Local Storage

## Technologies

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser Local Storage API

## Purpose of the Project

This project was developed as part of my learning process in **front-end development**.
The main goal was to build a practical application that demonstrates how JavaScript can control interface behavior, manage application state and store user data locally in the browser.
